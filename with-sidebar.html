When we want a sidebar, it should be fixed here. This area is only used for in-page navigation. It takes up the whole width of the page on smaller viewports (e.g. mobile devices). Examples of uses of this sidebar includes:

  • a list of filters for the search tool;
  • a table of contents for an information page that has a lot of questions, like the FAQs;
  • some sort of promotion.

We are showing how it stays to the left side of the page no matter how little or how much content there is in the main section on our right.

Do not use columns in this!
